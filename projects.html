Projects

Book Cover Classification

For this project, I sought to answer the question: "Can you classify a book into its respective genre or decade published based on the cover?" I found a dataset that included many different images of book covers along with the title, author, and genre information for each title. From there, I was able to gather the year the book was published. This image classification problem utilizes various neural net models to try to classify the cover image using the decade published and genre information.

Caffeine Analysis

This project was a solo hackathon that was completed in about 7 hours. I analyzed a dataset that had various types of drinks and their caffeine content, volume (ml), and calories. I used these features to predict the beverage type as belonging to energy drinks, coffee, soft drinks, tea, energy shots, and water.

Tanzania Waterpoint Analysis

This was a collaborative project analyzing water point functionality and water improvement in Tanzania. Clean drinking water is not accessible to much of the population, so we sought to find some potential improvement areas and discover which water points should be prioritized for improvement. Predictive modeling was used to determine water point functionality.

Reddit Classification & NLP

This project was about classifying posts from two different subreddits, and the two I decided to work with were r/harrypotter and r/Marvel. Using the Pushift API, I built a function to gather the data from each subreddit that was then used for the analysis and modeling. This binary classification problem utilized different classification models to determine which subreddit each post came from.
